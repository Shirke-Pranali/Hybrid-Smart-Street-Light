# Hybrid-Smart-Street-Light
Hybrid Solar-Wind Smart Street Light using Arduino

## Overview

This project is a smart street lighting system powered using renewable energy sources including solar, wind, and piezoelectric energy.

The system automatically switches lights ON/OFF using an LDR sensor and detects vehicles using an RCWL-0516 radar sensor.

---

## Features

- Solar Powered
- Wind Powered
- Piezoelectric Energy Harvesting
- Motion Detection
- Automatic Day/Night Detection
- Battery Charging

---

## Technologies Used

- Arduino Uno
- Arduino IDE
- C++
- RCWL-0516 Radar Sensor
- LDR
- TP4056
- MT3608
- 18650 Battery
